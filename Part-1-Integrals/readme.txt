		Integrals - Интегральный калькулятор

	Что может этот калькулятор:
	- Вычислять интегралы (y=...):
		1) x
		2) x+1
		3) корень из x (sqrt(x))
		4) x^2
		5) x^3
		6) sin x
		7) cos x
		8) корень из sin x (sqrt(sin(x)))
	- Численно находить определенные интегралы от произвольных
	  квадратурных функций: треугольников, трапеций и Симпсона.
	- Строить графики функций и квадратур.

	Как пользоваться калькулятором?

	- 1. Задайте числовые значения интеграла.
		
		В правой части программы Вы можете увидеть большой
		знак интеграла:

		*========================================*
		|      A                                 |
		|     //\\                    n = ...    |
		|     ||                                 |
		|     \\     (выражение) dx              |
		|       \\                               |
		|       ||                               |
		|     \\//                               |
		|      B                                 |
		*========================================*
		
		A - верхняя граница интегрирования
		B - нижняя граница интегрирования
		n - число разбиений

		Подставьте нужные вам числа в поля на месте букв.

	- 2. Выберите одну из тестовых функций.

		В правой нижней части программы, или под большим
		знаком интеграла, находится список доступных тестовых
		функций. 

		Нажмите на нужную вам тестовую функцию.

	- 3. Нажмите "РАБОТАТЬ".

		Появится график функции, построенный по вашему интегралу.
		В правом верхнем углу появятся числовые значения определенных
		интегралов по функциям Симпсона, трапеции и треугольника.

		Область графика имеет рабочие кнопки:
		
		Сверху справа - кнопки перемещения камеры по графику.
		Снизу справа:
			f(x) - показать/скрыть график функции f(x)
			Кс - показать/скрыть квадратуру по Симпсону
			Кт - показать/скрыть квадратуру по трапеции
			Кп - показать/скрыть квадратуру по прямоугольнику
			+/- - приблизить/отдалить

		!!! Калькулятор может выдать ответ "не число", но график
		!!! все равно будет отрисован.
